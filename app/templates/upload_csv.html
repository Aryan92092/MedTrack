CSV Upload Instructions

Upload a CSV file containing medicine data. The file should include the following columns:

Required Columns:

  • name - Medicine name (required)
  • quantity - Quantity in stock (required)
  • expiry_date - Expiry date in YYYY-MM-DD format (required)

Optional Columns:

  • category - Medicine category
  • manufacturer - Manufacturer name
  • batch_number - Batch number
  • purchase_price - Purchase price per unit
  • selling_price - Selling price per unit
  • min_stock_level - Minimum stock level (default: 10)
  • max_stock_level - Maximum stock level (default: 1000)

Example CSV Format:

name,quantity,expiry_date,category,manufacturer,purchase_price,selling_price
Paracetamol,100,2024-12-31,Analgesic,ABC Pharma,0.50,1.00
Aspirin,50,2024-11-15,Analgesic,XYZ Corp,0.75,1.50
Vitamin C,200,2025-06-30,Vitamin,Health Plus,0.25,0.75
Cancel

Upload Tips:

  • Make sure your CSV file is properly formatted with headers
  • Date formats supported: YYYY-MM-DD, DD/MM/YYYY, MM/DD/YYYY
  • If a medicine with the same name already exists, quantities will be added together
  • Empty optional fields will use default values
  • Maximum file size: 10MB
